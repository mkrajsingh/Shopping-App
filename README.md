# 🛒 Shoping site

## 📙 Description

Flikart is a project developed for online shopping inspired from ShopEase. It incorporates various technologies and libraries to achieve Html5, css3, React, typescript, redux, tailwindcss


# Clone the repository
https://github.com/mkrajsingh/Shopping-App/edit/main/README.md

# Navigate to the project directory
cd Shopping App


# Install dependencies
npm install
```

## Usage

To run the development server:

```bash
npm run dev
```

To build the project:

```bash
npm run build
```

To run tests:

```bash
npm test
```

## 🔑 Key Features

  - Add-to-Cart
  - Dark Mode
  - Header + Dropdown menu
  - Image slider
  - Product Details
  - Login Section
  - Order & Wishlist section


## 👨‍💻  Technologies Used

- **Frontend:**
  - React
  - Redux Toolkit
  - Tailwind CSS
  - Apollo Client
  - React Router Dom
  - GraphQL
  - Jest

- **Dev Dependencies:**
  - TypeScript
  - ESLint
  - Vite
